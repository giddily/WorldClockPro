# WorldClockPro

WorldClockPro is a Windows 11 world clock client designed for users who need to track multiple time zones clearly and efficiently.

## Features

- Modern Windows 11-style desktop interface
- Default home clock with customizable time zone
- Add, remove, edit, favorite, and reorder time zone cards
- Drag-and-drop sorting for clock cards
- Analog clock with real-time moving hands
- Digital time display for each city
- Day/night status, UTC offset, date, sunrise, and sunset information
- Searchable global time zone list
- Beijing display alias support
- English and Chinese language switching
- Light and dark theme support
- One-time activation license system
- Windows installer with custom installation directory support

## License Activation

WorldClockPro uses a one-time activation system. After activation, the license is saved locally on the current device for permanent use.

Each activation code is bound to one device.

## Supported Platform

- Windows 11

## Installation

Download the latest installer from the release page, run the setup file, choose an installation directory, and start using WorldClockPro after installation.

## Use Cases

- Remote work across multiple regions
- Cross-border business communication
- Overseas study and travel planning
- Tracking family, clients, or teams in different cities
- Quickly comparing time differences between countries
